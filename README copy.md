# ml3-lr2 Mochalov Anton

## Порядок установки проекта

1. Создать виртуальное окружение:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   ```

2. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Установить pre-commit:
   ```bash
   pre-commit install
   ```
